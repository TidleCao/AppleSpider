## 资源搜索

---

### 目标：

实现macOS/iOS 资源的快捷查找。

### 模块介绍：

-  Spider.app 基于macOS 10.11 平台的独立搜索软件
-  sniffer.xlpulgin 基于mac迅雷3.2.0以及后续版本的插件
-  iSpider 基于iOS11 平台的独立资源搜索软件
